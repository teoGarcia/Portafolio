SpaVecGenerator

SpaVecGenerator es un software realizado en el lenguaje de programación Java utilizando el gestor de Base de datos MySQL.
Este Proyecto fue realizado e implementado para que los estudiantes de Ingeniería Civil de la Universidad Politécnica "Manuela Saenz" pudieran ver puntos, vectores, segmentos y angulos en 3 Dimensiones pudiendo identificar las posiciones en las coordenadas X, Y y Z. Por otra parte tambien permite Guardar y Abrir Proyectos

Benso SPA

Benso S.P.A. es un software realizado en el lenguaje de programación Java utilizando el gestor de Base de datos MySQL.
Este fue un proyecto realizado para una Pequeña Constructora llamada Benso S.P.A. en la Región de Valparaiso, Chile y cuenta con 5 Modulos en los cuales se pueden nombrar los siguiente: Clientes, Cotizaciones, Compras, Ordenes de Compras y Proyectos.
Fué un proyecto 100% adaptado a las exigencias del cliente cumpliendo con lo esperado por el mismo
